# Consistencss

An "atomic css" style toolkit for [React Native](https://reactnative.dev/), inspired by [tailwindcss](https://tailwindcss.com/docs/installation/)

## Installation

```sh
npm install consistencss
```

using yarn:

```sh
yarn add consistencss
```
